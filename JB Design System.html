06 · Illustration Brief — Hand-Drawn Style
Style Direction

Hand-drawn / sketchy — think Mailchimp's "Freddie" era or Basecamp's illustration style. Rough, imperfect, warm strokes. Slight line variation, textured hatching. Characters should be abstract and inclusive — no photorealistic faces.

Colour Palette for Illustrations
#FFB300
#FFD54F
#FFFFFF
#1A1A1D
#52525A
#E65100
Stroke Style
Stroke weight: 2–3px, slightly variable
Fills: flat colour + optional hatching at 45°
No drop shadows — use the dark bg for depth
Line ends: slightly rounded caps
Imperfections welcome — not vector-perfect

Flat Shadow Rules
✓ Shadows are always hard offset (x y 0) — no blur radius
✓ Amber offset shadows on interactive / accent elements only
✓ Hover = translate(-2px,-2px) + bigger shadow
✓ Active/press = translate(4px,4px) + no shadow
✗ No diffuse / blurred shadows
✗ Don't mix offset directions — always top-left lift
✗ Limit bold amber borders to 1 element per section
Animation Principles
Transitions: 120–180ms ease
Hover lift: translate(-2px, -2px) + bigger shadow
Active press: translate(4px, 4px) + no shadow
Coin select: scale(1.1) burst
Nomination sent: Lottie confetti
Page transitions: slide up 300ms
For Claude Code
Copy the :root variables block into your globals.css.
Use .flat, .flat-amber, .flat-heavy classes.
Bold offset shadows in --shadow-amber-* and --shadow-*.
All components shown above are production-ready HTML+CSS — no framework required.
React components should use style props or CSS Modules mirroring these token names.
